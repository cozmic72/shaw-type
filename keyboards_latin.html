Shaw Type supports four different Shavian keyboard layouts, each with its own design philosophy and learning curve.

Shaw Imperial

The Shaw Imperial layout organizes characters by finger reach, with the most common phonemes on the home row. This is a popular choice for dedicated Shavian typists.

Learn more about Shaw Imperial →

Shaw QWERTY

Based on the familiar QWERTY layout, this makes it easier for beginners to transition to Shavian while keeping their existing muscle memory.

Learn more about Shaw QWERTY →

Shaw 2-layer (Shift)

A compact layout that uses the shift key to access the full character set, minimizing finger travel.

Learn more about Shaw 2-layer →

Shaw-JAFL

The "Just Another Friggin' Layout" - an experimental layout optimized for efficiency and comfort.

Learn more about Shaw-JAFL →

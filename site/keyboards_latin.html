Shaw Type supports a number of different Shavian keyboard layouts, each with its own design philosophy and learning curve. You will need to install them yourself to use the app for now - in the future we will add a virtual keyboard. Maybe.

In the mean-time, visit shavian.info/keyboards to get your hands on Shavian.info's Keyman keyboard layouts - they are the best.

Shaw Imperial

Based on the original Shavian Imperial Good Companion typewriter. This version is the Shavian.info Keyman layout.

New Shaw Imperial

Based on the original Shavian Imperial Good Companion typewriter. The current version in this app is actually my own variation on the layout that I'm developing, with auto-ligatures and some slight differences compared to the Shavian.info one. I'll add more info on that in the near future.

Shaw QWERTY

Another Shavian.info layout, based on the familiar QWERTY layout, this makes it easier for beginners to transition to Shavian while keeping their existing muscle memory. Find more info here.

Shaw 2-layer (Shift)

A compact layout that uses the shift key to access the full character set, putting related characters on the same key, based on the Keyman layout of the same name.

Shaw-JAFL

The "Just Another Friggin' Layout" - Shavian.info's experimental layout optimized for typing efficiency using the carpalx tool.

Auto ligatures

Inspired by the JAFL Keyman layout, the app supports auto ligatures for the compound letters in software. That is, you can type 路 then 路 to create 路, 路+路 to get 路, and similarly to get 路, 路 and 路. Try it out, and you may find you will never want to turn back.

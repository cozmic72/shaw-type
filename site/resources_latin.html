Read Lexicon

The one true source of truth for Shavian spelling, and the source of the dictionary that the word lists in this app were generated from. A must-have resource for every Shaver out there. Visit it at readlex.pythonanywhere.com. Be sure to read the spelling principles page if you haven't done so yet!

Shavian.info

The comprehensive resource for all things Shavian. Features keyboard layouts, fonts, learning materials, and conversion tools at shavian.info.

I cannot stress enough how important a resource this is for the community. If you haven't already done so, you should stop doing what you are doing and go and follow every single link on that website before coming back here. It's the best.

Shavian.school

Such a pity that the domain expired, but luckily it's still on github at shavian-school.github.io.

This was how I learned to read and write Shavian. Can recommend! (Beware: the author is using idiosyncratic names for the letters, don't get too attached to them.)

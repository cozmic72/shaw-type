Keyboards

Shaw Type supports a number of different Shavian keyboard layouts, each with its own design philosophy and learning curve. You can use the virtual keyboards provided by this app, or install your own, for example using the shavian.info keyboards.

Shaw Imperial

Based on the original Shavian Imperial Good Companion typewriter. This version is the Shavian.info Keyman layout.

No extra ligatures, automatic ligatures off by default.

About

Shaw Type is a typing practice tool designed to help you learn and master typing the Shavian alphabet on your favorite keyboard layout.

I looked around for a tool for practicing typing Shavian. I didn't find one, so I sat down to do some vibe-coding with the amazing Claude.code, and in a matter of a couple of hours, he (it? they?) produced the web app you see before you. We're living in the future, and we're all doomed.

In version 2.0, I found myself scratching another itch: the ability to practice typing using my iPad's magic keyboard. The new virtual keyboard support gives me just that.

Please give it a spin! Let me know on reddit what you think of it, and don't be shy with feature requests! Right now, it assumes that you are using a shavian keyboard of your own, physical or virtual (The Keyman layouts on Shavian.info are a solid choice for that.) I expect Claude will help me add a virtual keyboard in-app in the near future.

Change log

Version 2.0
  • Added in-app virtual keyboards: now you can use the app with QWERTY keyboards, and Shaw Type's virtual keyboard will take care of the rest!
  • Virtual on-screen keyboards! Great if you are using the virtual keyboard or a physical one without legends.
  • Vastly improved handling of automatic ligatures, verified in Safari and Chrome.
Version 1.3
  • Support alternative ligature input methods (VS1, JAFL).
  • Fixed timer statistics to track entire game session.
  • Improved lesson completion dialog with "Next lesson" button.
  • Added keyboard navigation for dialogs (Esc/Enter/Tab).
  • Dynamic font sizing for longer words.
  • Several small UX improvements.
Version 1.2
  • Fixed input handling for keyboards using input replacement.
Version 1.1.1
  • Fixed ligature mode - turning it off had no effect.
  • Fixed ligature mode handling in error highlighting and input blocking.
Version 1.1.0
  • UX improvement: countdown before game starts
  • UX improvement: block further user input if previous character was wrong
  • Embed and use Inter Alia and Ormin fonts
  • Fix Shavian spelling mistakes

Welcome to Shaw Type V2.0!

Shaw type is a typing practice tool for the Shavian alphabet, an alternative way of spelling the English language.

What's New

  • In-built virtual keyboard. No need to have a shavian keyboard yourself (physical or virtual), just use any old QWERTY keyboard - Shaw Type takes care of the rest!
  • Interactive virtual on-screen keyboard - see your chosen keyboard layout on screen!
  • Vastly improved handling of automatic ligatures: verified in Safari and Chrome - please let me know how you get on with it in your browser of choice!.
